FEATURE DIMENSIONS
==================
Input Shape: (max_seq_len, feature_dim) = (60, 126)
- max_seq_len: 60 frames (static signs use only first frame)
- feature_dim: 126 (2 hands × 21 landmarks × 3 coordinates)

Coordinate Structure:
- Hand 1: Landmarks 0-20 (63 features: x, y, z for each)
- Hand 2: Landmarks 0-20 (63 features: x, y, z for each)

AUGMENTATION SUMMARY
====================

Malayalam Static (0-6):
- Original: ~150 samples per class
- Techniques: POV horizontal flip
- Augmented: ~300 samples per class

Malayalam Dynamic (7-14):
- Original: ~5-10 valid samples per class
- Techniques: All 6 (noise, scale, translate, rotate, flip, temporal)
- Augmented: ~75 samples per class
- Note: 96% of original samples fail MediaPipe extraction

ISL (15-39):
- Original: ~500 samples per class
- Techniques: POV horizontal flip
- Augmented: ~1,000 samples per class
